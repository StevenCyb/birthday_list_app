# BDay
My first Flutter App to learn some fundamentals and to play around.
That means it might be buggy.

This App is aimed to help people to remember birthdays.

## Supported Platform
| Platform | Supported |
|----------|-----------|
| iOS      | ✅        |
| Android  | ✅        |
| Windows  | ❌        |
| macOS    | ❌        |
| Linux    | ❌        |
